# Tynkering

Public working repository for the Tynk app wireframe.

## Files

- `App.js` — the current approved Expo / React Native app wireframe.
- `package.json` — project setup and dependencies.
- `index.html` — a matching clickable browser preview for GitHub Pages.

## Turn on the clickable preview

1. Open this repository on GitHub.
2. Open **Settings**.
3. Select **Pages** in the left sidebar.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/(root)** folder.
6. Press **Save**.

GitHub will then show the public preview address on that Pages settings screen.

## Current approved flow

Splash → Welcome → Create account / Sign in / Explore → Home → Personal dashboard

Explore mode requires no account and does not permanently save changes.
